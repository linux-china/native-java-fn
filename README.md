Native Java function
====================

Native Java function with Graalvm, Kotlin and Fn project.

# How to build

* Make sure Docker is ready, and you can install Docker Mac.
* Install fn: brew install fn
* Build: fn build
* Run function
```
* fn run 
* echo -n "Fn" | fn run
```
